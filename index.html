Timepicker bootstrap-timepicker.js

About timepicker

A simple widget to easily select a time for an text input.

Download file

Live Demo

Default timepicker as component

Shown inside a modal with 24hr mode and seconds enabled.

Using bootstrap-timepicker

Call the timepicker:

$('.timepicker').timepicker({
    template : 'modal',
    showInputs: false
});

Markup

Display as component:

<div class="input-append bootstrap-timepicker-component">
    <input type="text" class="timepicker input-small">
    <span class="add-on">
        <i class="icon-time"></i>
    </span>
</div>

Options

Name type default description
template string 'dropdown' Specify the template to use. [dropdown, modal]
minuteStep integer 15 Specify a step for the minute field.
showSeconds boolean false Show the seconds field.
secondStep integer 15 Specify a step for the second field.
defaultTime string 'current' Set the initial time value. Setting it to 'current' sets it to the current time.
showMeridian boolean true 12hr / 24hr clock toggle.
showInputs boolean true Set to false to disable the text inputs in the widget.
disableFocus boolean false Disables the input from focusing. This is useful for touch screen devices that display a keyboard on input focus.
