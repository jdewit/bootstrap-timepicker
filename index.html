Using bootstrap-timepicker

Javascript:

$('.timepicker-default').timepicker();

HTML:

<div class="input-append bootstrap-timepicker-component">
    <input type="text" class="timepicker-default input-small">
    <span class="add-on">
        <i class="icon-time"></i>
    </span>
</div>

Javascript with callback event

$('.timepicker-default').timepicker()
    .on('changeTime', function(ev) {
      alert('time has changed')
      });
